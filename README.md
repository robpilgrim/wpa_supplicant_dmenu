
Dependencies: 
* dmenu
* wpa_supplicant
* iw
* dhcpcd

How to use:
1) Put the script in your PATH
2) Bind it to some key
3) Press the key

[Example](https://youtu.be/sJRGKM8RmXk)

Don't forget to run the script as root
